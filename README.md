🌟 FocusFlow: A 3D Animated Smart Study Dashboard

-------------------------------------------------------------
📖 Project Description

FocusFlow is a single-page, web-based productivity dashboard designed to help students manage tasks efficiently while maintaining cognitive focus.

Even though it’s built as a single HTML file, it leverages advanced CSS and JS animations to create a premium, 3D-like interactive experience. The interface is designed with Human-Computer Interaction (HCI) principles, including:

User-centered design
Usability heuristics
Mental model alignment

The goal is to provide a distraction-free, intuitive, and visually engaging interface that enhances productivity and maintains focus.

-------------------------------------------------------------

🎯 Features
Login / Signup Page
Simple entry with validation and visual feedback
Task Dashboard
Add, view, and prioritize tasks in floating 3D-style cards
Focus Mode
One-task-at-a-time view with timer
Smooth 3D transitions, background parallax, and subtle floating effects
Progress Tracker
Animated charts and cards showing weekly task completion
Premium Animations
Layered 3D cards
Hover lift and depth effects
Subtle parallax on mouse movement
Smooth transitions for page changes
Distraction-Free Mode
Minimal interface to maintain focus
Subtle background animations and breathing effects

-------------------------------------------------------------
🧠 HCI Principles
User-Centered Design: Clear layouts, visible actions, minimal distractions
Usability Heuristics: Feedback on actions, smooth transitions, error prevention
Mental Model Alignment: Focus mode mirrors how users expect to work on one task
Cognitive Load Reduction: Minimal interface and subtle motion enhance focus

-------------------------------------------------------------
🌌 Technology Used
HTML5 – Single-page structure
CSS3 – Animations, 2.5D/3D effects, gradients, shadows, glassmorphism
JavaScript – Interactivity, timers, micro-animations, task handling
Optional Libraries: (if used) Three.js, Anime.js, or Vanilla JS for animations

-------------------------------------------------------------
🚀 Getting Started
How to Run
Clone or download the repository
Open index.html in a modern browser (Chrome, Firefox, Edge)
Enjoy the 3D animated, interactive FocusFlow dashboard

No server or backend is required—all functionality is client-side
